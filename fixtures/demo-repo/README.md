# Acorn Tasks

A deliberately tiny Codeville demonstration project.

`summarizeProject(tasks)` in `src/health.js` is unfinished. Implement it without changing the tests, then run `npm test`.

Rules:

- `total` is the number of tasks.
- `complete` counts tasks with status `done`.
- `blocked` counts tasks with status `blocked`.
- `status` is `quiet` for no tasks, `blocked` when any task is blocked, `complete` when every task is done, and `active` otherwise.

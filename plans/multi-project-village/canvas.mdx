<Annotation id="stable-map-note" title="Stable spatial memory" targetId="town-overview" placement="bottom">

All five lots stay in place while the right panel changes selection. A project never expands into a replacement full-screen scene.

</Annotation>

<Annotation id="debrief-note" title="Two-layer debrief" targetId="completion-debrief" placement="bottom">

The small in-world bubble rewards completion; the selected-project panel preserves the same accessible text and follow-up action.

</Annotation>

<Annotation id="approval-note" title="Concurrency stays explicit" targetId="approval-queue" placement="bottom">

An approval pauses only its owning project. The queue retains project identity while unrelated builders continue.

</Annotation>
